# EB-Clean

## Overview

This repository contains the source code of the **Active Syntax** portfolio website.  
The site serves as a central, production-grade reference showcasing software development experience, technical competencies, and selected project implementations, including landing pages, websites, custom software solutions, and original open-source developments.

The primary goal of the project is to demonstrate **end-to-end technical capability**, covering architecture, frontend engineering, deployment, infrastructure configuration, and visual design within a single, coherent solution.

---

## Project Description

The website is implemented as a modern, fully responsive **single-page application** built with **Angular** and **Bootstrap**, following a modular, component-based architecture. Emphasis was placed on maintainability, clarity, and clean separation of concerns.

The application is deployed as a static site on **AWS S3** and globally distributed via **Amazon CloudFront**, ensuring low latency and high availability. Secure communication is enforced using **HTTPS with SSL/TLS**, with DNS records and CDN integration configured manually.

All **UI/UX design, layout, branding, logo, and visual assets** were created specifically for this project. Graphic assets were produced using **GIMP**, **Inkscape**, and AI-assisted tools in a **Linux-based development environment**.

The project reflects a full ownership approach: from concept and design through implementation, deployment, and infrastructure configuration.
